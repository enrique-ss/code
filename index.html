Sobre o Code Quest

Code Quest é uma visual novel interativa projetada para ensinar conceitos fundamentais de lógica de programação.

Ao longo da jornada, você assume o controle de um desenvolvedor transportado para um universo de fantasia medieval.

Use o Console JSON para ler a estrutura de objetos do mundo e escolha as opções corretas para resolver problemas e avançar na história!
